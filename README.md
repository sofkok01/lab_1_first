Отчёт по практической работе №1

"Типы данных и их внутреннее представление в памяти"

Цель работы

Изучить внутреннее представление основных типов данных в памяти компьютера, освоить работу с битовыми операциями и научиться манипулировать отдельными битами данных.

Задание

Определить размеры в байтах для различных типов данных.
Вывести двоичное представление целого числа (int).
Вывести двоичное представление числа с плавающей запятой (float).
Вывести двоичное представление числа двойной точности (double).
Реализовать возможность изменения любого бита в числе с использованием битовых операций.
Выполнить индивидуальное задание (вариант 9): обмен значений 3-го и 7-го битов (нумерация с 0 от младшего разряда).
Реализация

1. Определение размеров типов данных

Используется оператор sizeof() для определения размера в байтах:

cpp
void sizeOfType() {
    cout
        << "int: " << sizeof(int) << " байт" << endl
        << "short int: " << sizeof(short int) << " байт" << endl
        << "long int: " << sizeof(long int) << " байт" << endl
        << "float: " << sizeof(float) << " байт" << endl
        << "double: " << sizeof(double) << " байт" << endl
        << "long double: " << sizeof(long double) << " байт" << endl
        << "char: " << sizeof(char) << " байт" << endl
        << "bool: " << sizeof(bool) << " байт" << endl;
}
2. Вывод двоичного представления int

Знаковый бит выделен отдельно.
Остальные биты сгруппированы по 8 (байтам).
3. Вывод двоичного представления float

Структура: знак | экспонента (8 бит) | мантисса (23 бита).
Использован union для интерпретации памяти.
4. Вывод двоичного представления double

Структура: знак | экспонента (11 бит) | мантисса (52 бита).
Использован unsigned long long для доступа к 64 битам.
5. Изменение произвольного бита

Пользователь выбирает тип данных, номер бита и новое значение. Изменение выполняется через:

|= для установки бита в 1.
&= ~ для сброса бита в 0.
6. Индивидуальное задание (вариант 9)

Функция changeBit():

Считывает целое число.
Извлекает 3-й и 7-й биты.
Если они различаются, меняет их местами с помощью XOR.
Выводит результат в десятичном и двоичном виде.
